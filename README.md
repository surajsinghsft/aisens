
# 🚀 AI Career Coach

AI Career Coach ek smart web app hai jo users ko help karta hai:
- Resume build karne me
- ATS score check karne me
- Mock interview practice karne me
- AI se career guidance lene me

---

## 🧠 Features

✅ AI Resume Builder  
✅ ATS Score Checker  
✅ Mock Interview Questions  
✅ Career Suggestions using AI  
✅ User Authentication (Clerk)  
✅ Modern UI (Next.js + Tailwind)

---

## 🛠️ Tech Stack

- Frontend: Next.js, React
- Backend: Node.js, Server Actions
- Database: Prisma + PostgreSQL
- Authentication: Clerk
- AI: OpenAI / OpenRouter API
- Styling: Tailwind CSS
